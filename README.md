Quix
=================

Quix, provides an interface to remotely execute scripts to fix issues on unix servers



Requirement
-------------

  * brew install  mysql
  * easy_install pip
  * pip install flask
  * pip install sqlalchemy
  * pip install pyzmq
  * sudo -E pip install MySQLdb

On Mac OsX > 10.8
  * export CFLAGS=-Qunused-arguments
  * export CPPFLAGS=-Qunused-arguments
